## Formatear el texto

Hay varios caminos para formatear el texto en su salida, no los veremos todos, pero s√≠ los m√°s importantes.

El operador % se usa para formatear una tupla, y juntos formar un String. Pero necesita que se le indique antes qu√© tipo de datos vas a a√±adir:

* %s -> String
* %d -> Integer
* %f -> Decimal
* %.numero de d√≠gitos + f -> Decimal con cantidad decimal a mostrar.

```Python
name = 'davibern'
age = 333
output = 'Hi there, I am %s and I am %d years old' %(name, age)
print(output)
```

Si te fijas, hemos creado 3 variables, dos contienen datos y la √∫ltima la concatenaci√≥n de ambas. Adem√°s, esta √∫ltima, usa el operador % para concatenar dos datos diferentes, un String y un Integer.

Es tan potente, que podemos incluso concatener ya una colecci√≥n de datos ya creada.

```Python
books_reads = ['The Lord of the Rings', 'The Witcher', 'Game of Thrones']
output = 'I have read the following books' %(books_reads)
print(output)
```

Pero esta forma de **formatear** est√° en desuso, actualmente para las versiones 3.X, la que se est√° usando es la *String Interpolation*, o com√∫nmente llamada *f-String*.

> Se recomienda usar f-String por su facilidad de uso

Para crear una f-String en una salida incluimos una f, seguida de las comillas, y luego del texto. 

Aquellas variables que vayamos a incluir se escriben con llaves {} y dentro el nombre de la variable.

```Python
name = 'Crist√≥bal Col√≥n'
discover = 'The New World: The Americas'
print(f'{name} discovered {discover}, long time ago')
```

## Las cadenas de texto son secuencia de caracteres

En Python, como en cualquier otro lenguaje, el String se considera un tipo primitivo, porque representa el texto en su forma m√°s simple.

Pero si escarbamos un poquito, una cadena de texto no es m√°s que la concatenaci√≥n de letras. Y estas letras se llaman car√°cter, que en ingl√©s se representa como *char*.

El char es el tipo m√°s b√°sico de texto porque s√≥lo representa una unidad de texto, como por ejemplo 'a', o 'A'.

Tenemos adem√°s una tabla con la representaci√≥n del texto y su valor en bytes. ¬øTe suena [ASCII](https://es.wikipedia.org/wiki/ASCII)? Seguro que s√≠ üòâ

Y podemos probar lo que decimos desempaquetando una cadena de texto.

```Python
# Cadena de texto

# Desempaquetamos texto guardando cada posici√≥n en una variable
a, b, c, d = texto
print(a) # Imprime h
print(b) # Imprime o
print(c) # Imprime l
print(d) # Imprime a
```

## Accediendo a la posici√≥n de un car√°cter

En programaci√≥n, por lo general, la primera posici√≥n de un vector empieza siempre en 0. Hay algunas excepciones, como [R](https://www.r-project.org/), en el que empiezan en 1. Pero no es lo habitual.

Si volvemos a nuestra variable anterior y escbribimos:

```Python
texto = 'hola'
print(texto[0]) # -> 'h'
print(texto[1]) # -> 'o'
print(texto[2]) # -> 'l'
print(texto[3]) # -> 'a'
```

Vemos como estamos accediendo, empezando por 0, a la posici√≥nd de cada car√°cter.

¬øY si queremos acceder la √∫ltima posici√≥n?

Te dejo a continuaci√≥n dos m√©todos.

<u>Contar y luego acceder</u>

Si no sabemos la longitud de una cadena de texto, lo mejor es calcularle su longitud.

Luego, al total se le resta 1, porque las cadenas de texto internamente siempre se guardan con un espacio a su final.

```Python
planet = 'Earth'
planet_length = len(planet)
print(planet[planet_length - 1]) # Devuelve 'h'
```

<u>Usar el s√≠mbolo negativo</u>

Si queremos acceder de forma mucho m√°s f√°cil usamos el s√≠mbolo negativo para acceder a la √∫ltima posici√≥n.

```Python
planet = 'Earth'
print(planet[-1]) # Devuelve 'h'
```

Si quisi√©ramos acceder a la pen√∫ltima, ser√≠a un -2, y as√≠ sucesivamente.

Lo que antes nos ha ocupado 3 l√≠neas, aunque se puede reducir, con esta forma de Python lo hemos hecho en una l√≠nea.

## Funciones de la clase String

String, como recordatorio, es una clase que maneja la concatenaci√≥n de caracteres.

Y como toda clase que se precie, tiene sus atributos y m√©todos. M√°s adelante veremos qu√© es una clase, pero por ahora, tienes que saber que String tiene una serie de "herramientas" para poder jugar con el texto.

### capitalize()

Convierte la primera letra de una frase en may√∫scula.

```Python

```

### count()

Cuenta el n√∫mero de ocurrencias de una cadena de texto. Es decir, le indicas el car√°cter a buscar y devuelve las veces que existen. 

Se puede incluso focalizar en una parte del texto.

```Python

```

### endswith()

Comprueba si la parte final de un string coincide con lo que indiques en sus argumentos.

```Python

```
